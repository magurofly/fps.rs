# コピペで使える FPS

## 機能

- 四則演算
  - `f + g`, `f - g`, `f * g`: 和・差・積
  - `f / g`: 逆元をかける（結果の次数は `f`, `g` のうち大きい方に合わせる）
- 逆元
  - `f.inv()`: 逆元（次数は `f` と同じ）
  - `f.inv_n(n)`: 逆元（次数は `n-1` ）

## 使い方

1. [minified.rs](./minified.rs) の内容をコピペしてコードに貼り付ける
2. mod に応じて `use fps::{FPS998244353 as FPS, ModInt998244353 as Mint};` などを書く
